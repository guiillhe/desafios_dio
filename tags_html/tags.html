Guia Rápido de Tags HTML

Esta página serve como um guia de referência para algumas tags HTML essenciais e de formatação de texto.

Títulos: <h1> até <h6>

As tags de <h1> a <h6> são usadas para definir títulos e subtítulos em uma página. A tag <h1> representa o título mais importante e a <h6>, o de menor importância.

Parágrafo: <p>

A tag <p> é usada para definir um parágrafo de texto. Navegadores automaticamente adicionam um espaço antes e depois de cada elemento <p>.

Este é um exemplo de um parágrafo. Ele agrupa sentenças relacionadas, facilitando a leitura do conteúdo.

Formatação de Texto

Ênfase Forte: <strong>

A tag <strong> é usada para dar ênfase forte a um texto, que geralmente é exibido em negrito pelos navegadores.

Itálico: <i>

A tag <i> é usada para colocar uma parte do texto em itálico. Geralmente utilizada para termos técnicos, nomes de navios, etc.

Sublinhado: <u>

A tag <u> renderiza o texto sublinhado. Seu uso deve ser cuidadoso para não ser confundido com um link.

Texto Marcado: <mark>

A tag <mark> é usada para destacar ou marcar uma parte do texto para referência.

Texto Pequeno: <small>

A tag <small> define um texto menor, como direitos autorais ou notas de rodapé.

Texto Deletado: <del>

A tag <del> é usada para representar um texto que foi removido de um documento.

Subscrito e Sobrescrito: <sub> e <sup>

A tag <sub> define um texto subscrito, como em H2O. A tag <sup> define um texto sobrescrito, como em X2.

Fonte (Obsoleta): <font>

A tag <font> era usada para definir o tamanho, a cor e a fonte do texto. Exemplo. Atenção: Esta tag é obsoleta em HTML5 e seu uso não é recomendado. A estilização deve ser feita com CSS.

Listas

Lista Ordenada: <ol> e <li>

A tag <ol> define uma lista ordenada (numerada). Cada item da lista é definido pela tag <li>.

  1. Primeiro item
  2. Segundo item
  3. Terceiro item

Lista Não Ordenada: <ul> e <li>

A tag <ul> define uma lista não ordenada (com marcadores). Cada item da lista também é definido pela tag <li>.

  • Item com marcador
  • Outro item com marcador
  • Mais um item

Outras Tags Importantes

Link (Âncora): <a>

A tag <a> define um hyperlink, que é usado para ligar uma página a outra. O atributo href especifica a URL de destino.

Este é um link para o Google.

Linha Horizontal: <hr>

A tag <hr> é usada para criar uma quebra temática no conteúdo, geralmente exibida como uma linha horizontal.

Citação em Bloco: <blockquote>

A tag <blockquote> especifica uma seção que é citada de outra fonte.

"Esta é uma citação. Navegadores geralmente indentam este elemento para destacá-lo do resto do texto."

Abreviação: <abbr>

A tag <abbr> é usada para definir uma abreviação ou um acrônimo. O atributo title pode ser usado para mostrar o significado completo ao passar o mouse sobre o elemento.
